# Portfolio Website

A modern and responsive personal portfolio website developed to showcase my skills, projects, freelance services, and professional experience.

## Overview

This portfolio represents my work as a:

- Full Stack Developer
- Graphic Designer
- Video Editor
- Content Creator
- Media Manager

The website features a professional design with animations, project showcases, service packages, and a contact section.
